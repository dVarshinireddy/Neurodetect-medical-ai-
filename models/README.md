# NeuroDetect AI Models

Model files are not stored in GitHub
due to file size limitations.

## Model Accuracies

| Model | Accuracy | Type |
|-------|----------|------|
| CNN | 98.87% 🏆 | Deep Learning |
| SVM | 95.95% | Machine Learning |
| RF | 94.23% | Machine Learning |
| VGG16 | 94.11% | Transfer Learning |
| ResNet50 | 85.83% | Transfer Learning |

## Dataset
- Source: Kaggle Brain Tumor MRI
- Images: 7,023 MRI scans
- Classes: 4 (Glioma, Meningioma,
           Pituitary, No Tumor)
- Split: 80% train / 20% test
