## 🎯 What is NeuroDetect?

NeuroDetect is a hospital-grade AI system
that detects brain tumors from MRI scans
using an ensemble of 6 AI models with
98.87% accuracy.

## 🤖 AI Models

| Model | Accuracy | Type |
|-------|----------|------|
| CNN | 98.87% 🏆 | Deep Learning |
| SVM | 95.95% | Machine Learning |
| RF | 94.23% | Machine Learning |
| VGG16 | 94.11% | Transfer Learning |
| ResNet50 | 85.83% | Transfer Learning |

- Classes: Glioma, Meningioma,
           Pituitary, No Tumor
